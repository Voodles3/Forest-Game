# Forest Game
### A singleplayer, first-person, adventure-horror game prototype created in Unity 2022.3.44f1.
Note: this is an unfinished project and does not currently have a complete game loop.

**Features:**
- Open world exploration
- Hotbar inventory; item pickup, drop, and selection
- Smooth 3D character movement

**Tech Stack:**
- *Engine:* Unity 2022.3.44f1
- *Language:* C#
- *Asset creation:* Blender, Blockbench
